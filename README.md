# Prodlog marketing site

Next.js marketing site for [Prodlog](https://prodlog.app).

## Run locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Optional: set `NEXT_PUBLIC_SITE_URL` in `.env.local` for canonical URLs and sitemap (defaults to `https://prodlog.app`). Set `NEXT_PUBLIC_OG_IMAGE` to override the default Open Graph image URL.
3. Run the dev server:
   `npm run dev`
4. Build for production:
   `npm run build`
5. Start the production server:
   `npm start`
